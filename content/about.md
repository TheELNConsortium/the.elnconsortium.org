# History

The idea of this consortium was born during the [HIDA-funded](https://www.helmholtz-hida.de/) stay of Nicolas Carpi at the [FZJ](https://www.fz-juelich.de/de) / [IEK-2](https://www.fz-juelich.de/iek/iek-2) in 2022.

# Joining the Consortium

We welcome new members. In order to join the ELN Consortium:

1. Open a Pull Request on https://github.com/TheELNConsortium/TheELNFileFormat/ adding your .eln example file in the `examples` folder.
2. We will then review your Pull Request to ensure the file is compliant to specifications
3. Once the PR is validated we will add you as an official member
